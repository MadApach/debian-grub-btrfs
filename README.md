## English

Welcome to the automation toolset for configuring an advanced, robust Btrfs filesystem setup tailored for Debian (Testing/Sid). This project provides a step-by-step solution to fix common bootloader bugs, optimize snapshots, and ensure seamless system recovery.

### Repository Structure

This repository is split into language-specific folders. Each folder contains a complete set of translated automation scripts and its own detailed documentation:

* 📁 **[English Version (en/)](./en/)** — Includes all 6 scripts fully localized in English along with a dedicated setup guide.
* 📁 **[Russian Version (ru/)](./ru/)** — Includes all 6 scripts with Russian terminal output and localization alongside its comprehensive guide.

Please navigate to the folder of your preferred language to read the complete installation manual and access the scripts.
